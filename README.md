# Machine-Learning-Based-Network-Intrusion-Detection-System
Proyek penelitian ini berfokus pada pengembangan model deteksi intrusi jaringan berbasis Machine Learning menggunakan dataset UNSW-NB15, NSL-KDD, dan CICIDS 2019. Model yang digunakan meliputi K-Nearest Neighbors, Random Forest, dan XGBoost, dengan evaluasi berdasarkan accuracy, precision, recall, F1-score, dan confusion matrix.
